# OpenSiliconLabs

OpenSiliconLabs is dedicated to the design, verification, and firmware bring-up of open-source custom silicon.

Our core focus is building a highly modular, foundational **RISC-V System-on-Chip (SoC)** that serves as a launchpad for diverse **ASIC implementations**. By co-designing the hardware architecture alongside the hardware abstraction layers (HAL) and bootloaders, we ensure seamless silicon bring-up and optimized performance.

### Core Initiatives

* **Base SoC Architecture:** Developing a flexible, verified RISC-V core designed for scalability.
* **Custom ASIC Spin-offs:** Leveraging the base SoC to architect specialized chips (e.g., edge compute, hardware acceleration).
* **Robust Firmware:** Writing generic, high-performance firmware to ensure reliable hardware-software integration from day one.
